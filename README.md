# Password Manager
Password Manager for Debian lets you securely access your saved passwords so you'll always remember what password to enter to access your account. All you need is Zenity and YAD to get started with Password Manager. To install Password Manager, download the "Installer.tar.gz" file onto your computer and execute the file.
[Password Manager.tar.gz](https://github.com/GabeCoder/password-manager/files/6256044/Password.Manager.tar.gz)
Once you've finished installation, you can use the menu to open Password Manager.
